# Family Realm

**Family Realm** is a web application designed to preserve family history and ancestral memory.
The platform allows families to create private or public spaces where relatives can collaborate to document family members, stories, events, and historical materials.

This project was developed as a **university diploma project**.

---

## Features

* User registration and authentication (JWT)
* Create **private or public family groups**
* Invite relatives to a family group via email link
* Create and manage **family members (living and deceased)**
* Store detailed information about each person
* Upload **documents, photos, and videos**
* Create and manage:

  * Family tree
  * Timeline
  * Family stories
  * Events
  * Family places
* Multimedia and document archive
* Public articles and archive available for non-registered users
* Search functionality

## Application Structure

Main sections of the platform:

* **Family Groups** – create and manage private or public family communities
* **Persons** – profiles of living and deceased family members
* **Family Tree** – visualization of family relationships
* **Timeline** – chronological family history
* **Family Stories** – important historical narratives
* **Events** – key moments in family history
* **Family Places** – locations connected to family history
* **Media Archive** – photos, videos, and documents

## Installation

Clone the repository:

```
git clone https://github.com/your-username/family-realm.git
```

Go to the project directory:

```
cd family-realm
```

Install dependencies:

```
bundle install
```

Setup the database:

```
rails db:create
rails db:migrate
```

Run the server:

```
rails server
```

The application will be available at:

```
http://localhost:3000
```
