**Family Realm** is a web application designed to preserve family history and ancestral memory.
The platform allows families to create private or public spaces where relatives can collaborate to document family members, stories, events, and historical materials.

This project was developed as a **university diploma project**.

---

## Features

* User registration and authentication (JWT)
* Create **private or public family groups**
* Invite relatives to a family group via email link
* Create and manage **family members (living and deceased)**
* Store detailed information about each person
* Upload **documents, photos, and videos**
* Create and manage:

  * Family tree
  * Timeline
  * Family stories
  * Events
  * Family places
* Multimedia and document archive
* Public articles and archive available for non-registered users
* Search functionality
